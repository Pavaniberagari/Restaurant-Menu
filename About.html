About Us

Spice Garden is a simple restaurant website project designed to showcase menu items, dish categories, pricing, and a clean user-friendly interface.


This project was built using HTML, CSS, JavaScript, and Python (Flask).


It helps users explore dishes, view prices, and book a table easily.
